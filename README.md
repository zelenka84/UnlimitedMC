# UnlimitedMC — лаунчер (v0.8, макет v6)

Кастомный лаунчер Minecraft (Java Edition): локальный профиль, моды из **Modrinth** и
**CurseForge** с **автоустановкой зависимостей**, загрузчики **Fabric / Forge / Quilt**,
**тёмная и светлая темы**, акцентный цвет → **монохромный градиент** (вживую, 5 свотчей),
логотип-знак, **реактивный ambient-фон**, **сворачиваемый по наведению сайдбар**,
**локализация RU/EN** и **поле прокси** (для РФ, где Modrinth заблокирован). Главная — с
случайным приветствием, новостями и страницей статьи.

Каждая сборка — со своим **менеджером** (моды / ресурспаки / шейдеры по файлам на диске,
миры с импортом-экспортом `.zip`, сохранённые сервера, галерея скриншотов, RAM на сборку)
и превью-**коллажем логотипов модов**. Каталог Modrinth/CurseForge — с **популярным по
умолчанию** и **постраничной навигацией**; иконки интерфейса — векторные (не эмодзи).


## Структура кода
- `unlimitedmc.py` — интерфейс и экраны (точка входа).
- `ui_kit.py` — дизайн-система: темы, акцент→градиент, логотип, ambient-фон, виджеты, QSS.
- `ui_content.py` — данные интерфейса: локализация, приветствия, новости.
- `umc_core.py` — ядро (установка/запуск игры, моды, прокси) без GUI.

## Что нужно
- **Windows** (также macOS/Linux)
- **Python 3.10+** — https://www.python.org/downloads/ (галочка «Add Python to PATH»)
- **Java** в системе (для запуска игры). Для современных версий — Java 17+.

## Установка
1. Распакуй папку `UnlimitedMC`.
2. В этой папке открой PowerShell и выполни:
   ```
   pip install -r requirements.txt
   ```
3. Запусти:
   ```
   python unlimitedmc.py
   ```

## Первый запуск
Появится короткий онбординг — впиши ник и выбери акцентный цвет. Дальше:
1. **Играть** → «Создать сборку» (версия + загрузчик).
2. **Modrinth / CurseForge** → выбери сборку в «Ставить в:», найди мод, «Установить».
   Обязательные зависимости мода ставятся автоматически; уже скачанное пропускается.
3. **Играть** → выбери сборку → «Запустить» (первый запуск качает игру — это долго).
4. **Менеджер сборки** (кнопка на карточке сборки) → настройки и RAM на сборку, контент
   (моды/паки/шейдеры с удалением), миры (импорт/экспорт), сервера, скриншоты.

## Прокси (важно для РФ)
Modrinth (и его файлы) в РФ заблокирован. Если поиск/установка не работают:
- **Настройки → Прокси** → впиши свой прокси: `http://host:port` или `socks5://host:port`.
- Прокси применяется ко **всему** трафику лаунчера, включая загрузку самой игры.
- Для **socks5** нужен пакет PySocks: `pip install pysocks`.

CurseForge обычно не заблокирован, но требует бесплатный ключ:
console.curseforge.com → **Настройки → CurseForge ключ**.

## Где данные
- Windows: `%APPDATA%\UnlimitedMC`
- macOS: `~/Library/Application Support/UnlimitedMC`
- Linux: `~/.local/share/UnlimitedMC`

## Чего пока нет
- **Аккаунты UnlimitedMC и вход** — нужен сервер, поэтому везде, где требуется вход
  (аккаунт-чип, «Сообщество», профиль, вход на umclaunch.net), стоит честная заглушка
  «Вход недоступен». Локальный игровой ник (офлайн, вкладка «Играть») с аккаунтом не связан.
- Вход через Microsoft / Ely.by — только локальный офлайн-ник.
- NeoForge — есть Fabric, Forge, Quilt.
- Часть модов CurseForge запрещает скачивание вне сайта — такие не установятся.
- Скачивание сборок (modpack) из каталога — ставим только моды, ресурспаки и шейдеры.
- Свой пользовательский значок сборки — превью собирается из логотипов её модов.
